Travaux pratiques

Il est conseillé de faire les TP sous Matlab. Sous Octave, pour que cela fonctionne, il faut installer les toolbox manquantes.

  • pkg install -forge control
  • pkg install -forge signal
Chaque installation semble prendre une dizaine de minutes. Ensuite pour utiliser le package signal, il faut faire
  • pkg load signal
Et sous Octave, il y a un problème qui n’est pas encore résolu : Il n’y a pas de fonction timer, et donc les programmes TP1_2.m, TP4.m et TP5.m ne fonctionnent pas. (Les fonctions déjà écrites et enregistrés avec une extension .p ne fonctionnent pas sous Octave)
